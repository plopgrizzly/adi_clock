Clock

use adi_clock::Clock;

Clock represents the state of a Real-Time-Clock (RTC). You can use it to make animations, time operations, or determine the date and time (TODO).

fn get() -> Clock

Get the current state of the Real-Time-Clock (RTC).

fn since(&self) -> f32

Get the number of seconds since self was initialized (or since get() was called).

use adi_clock::Pulse;

impl Pulse for f32

Call these functions on the return value of Clock::since(). Use rate_spr to specify how many seconds it takes to cycle through the animation.

fn half_linear_pulse(&self, rate_spr: f32) -> f32

The return value will be animated from 0 to 1 always moving the same amount for each interval.

fn full_linear_pulse(&self, rate_spr: f32) -> f32

The return value will be animated from 0 to 1 and back to 0 always moving the same amount for each interval.

fn full_smooth_pulse(&self, rate_spr: f32) -> f32

The return value will be animated from 0 to 1. The closer it is 0 and 1 the slower it will move, the farther away from 0 and 1 the faster it will move.

fn half_smooth_pulse(&self, rate_spr: f32) -> f32

The return value will be animated from 0 to 1 and back to 0. The closer it is 0 and 1 the slower it will move, the farther away from 0 and 1 the faster it will move.
